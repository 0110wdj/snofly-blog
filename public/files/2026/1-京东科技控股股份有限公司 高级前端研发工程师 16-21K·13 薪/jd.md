## 岗位职责

1. 负责 IM 前端技术选型、架构设计、开发实现及优化工作，确保前端性能与用户体验达到最优；
2. 对接业务需求，实现业务逻辑的前端展示与交互，确保产品功能的完整性和准确性；
3. 负责前端页面的性能调优，包括但不限于代码优化、资源加载优化、渲染优化等；
4. 持续关注前端技术发展趋势，将新技术、新方法引入项目中，提升项目的技术水平和竞争力。

## 岗位要求

1. 拥有 5 年及以上前端开发工作经验，熟悉 HTML5、CSS3、JavaScript 等基础技术，熟悉 W3C 标准及规范，有良好性能优化意识；
2. 掌握 React + Redux、Vue 等开发框架，熟悉常用数据结构和算法；
3. 掌握 Webpack 前端工程化流程，熟悉 Git 等前端协作工具；
4. 具有大型 IM 项目或协同办公平台开发经验者优先；有 Electron 开发经验者优先；
5. 对 Web 技术有浓厚兴趣，了解行业热门技术特点，具备持续学习意识，逻辑思维能力强；
6. 具备良好的沟通能力和团队协作精神，工作态度严谨、踏实，有较强的质量意识和抗压能力。
