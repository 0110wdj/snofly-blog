## 模拟面试问题清单（预计 25-40 分钟）

### 一、项目深度深挖（10-15 分钟）

#### 1. 虚拟滚动优化

**问题：** 你在菱电 TBOX 项目中优化了十万行大型表格的渲染性能，用了虚拟滚动。请你讲一下虚拟滚动的实现原理，你在实现过程中遇到了哪些坑，怎么解决的？

**追问：** 如果表格有不同行高怎么办？你怎么计算可渲染区域？
**追问：** 虚拟滚动会有空白区域滚动的时候闪烁，你怎么处理的？
**追问：** 除了虚拟滚动，你还做了哪些优化让渲染更快？比如 React 层面怎么缓存？

#### 2. Electron 项目迁移到 Web

**问题：** 你把一个 Electron 项目迁移到 Web 端，具体做了哪些工作？遇到了哪些兼容性问题？

**追问：** git submodule 是怎么管理两个代码库的？为什么不用 monorepo？
**追问：** Electron 里的 ipc 通信在 Web 里怎么处理的？
**追问：** 同时维护两套代码会不会有维护成本问题？你怎么平衡的？

#### 3. WebSocket 数据压缩优化

**问题：** 你自己写了 json-string-pack 工具，将 9MB 压缩到 500KB。讲讲你的压缩思路是什么？为什么不直接用 JSON.stringify 或者 protobuf？

**追问：** 为什么不确定结构用你这个方案，确定结构用 protobuf？
**追问：** 你的压缩算法核心是什么？是去掉重复的键吗？
**追问：** 解压那边怎么处理？性能怎么样？

#### 4. 前端开发规范设计

**问题：** 你引入了 TypeScript 和 ESLint 做规范，讲讲你是怎么推动团队落地这个规范的？怎么减少对现有开发的影响？

**追问：** git hook pre-commit 会不会影响提交速度？你做了什么优化？
**追问：** 逐步迁移类组件到函数组件，你是怎么做的？策略是什么？

### 二、技术基础 & 深度（8-12 分钟）

#### HTML/CSS

1. HTML5 新增了哪些特性？你在项目中用到过哪些？
2. CSS 盒模型是什么？box-sizing: border-box 和 content-box 的区别？

5. 什么是重绘和重排？怎么减少重排？

#### JavaScript

1. 讲一下对 JavaScript 闭包的理解？闭包有什么应用场景？
2. 原型和原型链是什么？能画个图说明一下吗？

#### React

#### 浏览器 & 网络

1. 从输入 URL 到页面渲染完成，整个过程发生了什么？
2. 浏览器缓存机制讲一下？强制缓存和协商缓存的区别？
